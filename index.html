Apolo is a lightweight, embeddable scripting language forged in C.

Designed to be simple, fast, and clear—shining a light on how interpreters work under the hood.

Start Coding

Installation

Apolo is distributed as pure C source code or as an .exe file. If you are not using Windows or wish to compile manually, you will need a standard C compiler.

1. Download Source

Clone the repository or download the source files into a folder.

2. Compile

Use the provided executable (Windows only) or compile manually with GCC/Clang (for Windows or any other system).

$ gcc main.c vm.c compiler.c scanner.c chunk.c value.c object.c table.c -o apolo

This will generate the apolo executable.

Running Code

REPL (Sun Shell)

Run without arguments to enter the interactive mode.

$ ./apolo
apolo > print "Dawn of a new code";
Dawn of a new code

Script Execution

Pass a file path to execute a script file.

$ ./apolo main.apolo

Variables & Types

Apolo is dynamically typed. Use var to declare new identifiers.

var rays = 100;
var star = "Sun";
var isBright = true;
var darkness = nil;

Data Types

Type Description
Number Double-precision floating point (e.g., 3.1415).
String Text strings enclosed in double quotes.
Boolean Logic values: true or false.
Nil Represents the absence of a value.

Input & Output

Print

Outputs text or numbers to the console.

print "Hello, World";
print 10 * 10;

Input

Pauses execution to get a line of text from the user.

print "Enter your name:";
var user = input();
print "Welcome, " + user;

Control Flow

If / Else

var temp = 30;

if (temp > 25) {
    print "It is sunny.";
} else {
    print "It is cold.";
}

While Loop

var count = 5;
while (count > 0) {
    print count;
    count = count - 1;
}
print "Liftoff!";

Comments

Lines starting with # are ignored by the compiler.

# This is a comment about the sun
var light = true; # Inline comment
