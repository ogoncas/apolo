About Apolo

Apolo is a lightweight, interpreted programming language designed for simplicity, performance, and ease of use. Drawing inspiration from the Greek god Apollo—symbol of light, knowledge, and the arts—Apolo aims to bring clarity and elegance to scripting and rapid development.

Version: 2.0 | Date: December 19, 2025

Key Features

  • Dynamic typing supporting numbers, booleans, strings, and nil values.
  • Local and global variables with proper scoping.
  • Control structures including if-else statements and while loops.
  • Arithmetic, comparison, and logical operators.
  • Built-in I/O functions: print and input.
  • Interactive REPL for quick testing and experimentation.
  • Efficient string interning and hash table implementations.
  • Robust error handling for both compilation and runtime issues.

Documentation

Apolo's syntax is intuitive and beginner-friendly. Below is a quick reference guide:

Declaring Variables

var x = 10;
var name = "Apolo";

Control Flow Structures

if (x > 5) {
    print "Greater than 5";
} else {
    print "5 or less";
}

while (x > 0) {
    print x;
    x = x - 1;
}

Input and Output

var userInput = input();
print "Welcome, " + userInput;
